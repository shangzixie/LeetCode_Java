# 刷题技巧-1

## 将二维数组转换为一维数组

```
              n
      ____________________
     |                    |
     |                    |
 m   |                    |
     |                    |
     |                    |
      ____________________
```

对于一个二维数组`grid`, 其中 `len(grid)` 是`m`, `len(grid[0])` 是 `n`, 可将其转换为一维数组 `arr`, 长度为`m * n`. 每一个点`grid[i][j]`对应一维数组`arr[i * n + j]`
